# FAF - Frequency-Altered Auditory Feedback

Андроид-приложение для терапии заикания, использующее метод FAF (Frequency-Altered Auditory Feedback) — технику изменения частоты слуховой обратной связи, которая помогает улучшить плавность речи.

## ⚠️ Важно

Это приложение является **дополнительным инструментом** и не заменяет работу с логопедом. Для достижения наилучших результатов необходима комплексная терапия под руководством специалиста.

## О разработке

Программа создана при поддержке и консультациях опытного логопеда для взрослых, который специализируется на:
- Коррекции заикания
- Постановке звуков
- Восстановлении речи после инсульта
